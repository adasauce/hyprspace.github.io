Why Hyprspace?

Minimal Install

Install Hyprspace as a single 22MB binary on your system.

Simple Setup

Hyprspace allows computers anywhere in the world to connect to each other without port forwarding, or having to know their IP Addresses in advance.

Truly Distributed

All computers in a Hyprspace network connect directly to each other so there's no single point of failure.
